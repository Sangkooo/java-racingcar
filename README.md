# 자동차 경주 게임

## 기능목록 !

- [x] `자동차`는 이름을 가진다.
- [x] `자동차`는 여러대 일 수 있다.
- [x] `자동차`들은 이름의 길이가 5자 미만이다.
- [x] 9이하의 `난수`를 생성 한다.
- [x] `자동차`는 9이하의 난수 중 4 이상이 되면 전진한다.
- [x] `자동차`는 9이하의 난수 중 4 미만이 되면 멈춘다.
- [x] `자동차`가 전진할 때 "-"로 표시된다.
- [x] 경기 횟수를 `입력` 받는다.
- [x] 매 경기마다 전진 혹은 멈추는 것이 `표시`된다.
- [x] 마지막 `경기` 때 `위치`값이 가장 높은 자동차가 우승자이다.
